# Workflows del Método Ceiba

Esta carpeta contiene los workflows de procesos de desarrollo.

Los workflows se crearán según las necesidades del equipo usando *create-workflow.

## Estructura Sugerida

Cada workflow debe tener su propia carpeta con:
- workflow.yaml (configuración)
- instructions.md (pasos del workflow)
- template.md (si genera documentos)
